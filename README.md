# AI Chat Application

> This project is based on an open-source tutorial and serves as a learning foundation.
> I am actively enhancing, refactoring, and extending it with new features, improved architecture, and production-ready practices as
  part of my full-stack development journey.

## Overview

A full-stack real-time AI-powered chat application built using the MERN stack. The application enables users to communicate through a modern chat interface with authentication, real-time messaging, and AI-assisted conversations.

This repository is continuously updated with new features, UI improvements, performance optimizations, and code refactoring.

## Getting Started

Clone the repository

```bash
git clone <your-repository-url>
```

Install dependencies

### Backend

```bash
cd Backend
npm install
```

### Frontend

```bash
cd Frontend
npm install
```

Start the backend

```bash
npm run dev
```

Start the frontend

```bash
npm start
```
